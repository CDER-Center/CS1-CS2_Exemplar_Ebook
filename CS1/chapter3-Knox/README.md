By way of context, Knox uses an academic calendar where each term is slightly less than 10 weeks.
The CS1 class meets 4 times a week, 3 lectures (MWF) and a lab (Tu this term).
All class meetings are 70 minutes long.
Before many classes, students have reading assignments from the [Runestone](https://runestone.academy/) textbook
Introduction to Programming with Java (CS Java).
The book includes interactive exercises and the students receive participation points for completing each assignment.
This course uses Peer Instruction, a pedagogy where lecture is interrupted with multiple-choice questions.
The students answer these individually, then discuss them in groups and re-answer.
Then the instructor reveals the correct answer, has a student give an explanation of this answer, and then
fills in any additional material.
These questions are graded only for participation.
Students are expected to complete the submit the lab assignments; the lab period is a time for them to get started,
with the instructor and a TA available to answer questions.
Then they continue on the assignment, which is due at the beginning of the following week's lab period.
