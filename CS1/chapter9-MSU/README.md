# CSIT 111 – Fundamentals of Programming I

This repository contains course and instructional materials developed for
CSIT 111 at Montclair State University. The course package integrates parallel
and distributed computing (PDC) concepts into a CS1 course through the
**Flag Maker** unplugged activity.

## Course Information

- **Institution:** Montclair State University (MSU)
- **Course number and title:** CSIT 111 – Fundamentals of Programming I
- **CS1 or CS2:** CS1
- **Programming language:** Java
- **Main unplugged activity:** Flag Maker
- **Main plugged activities:** None
- **E-book chapter:** Chapter 10: CS1 Course Package – Montclair State University (MSU)
- **Email contact:** [wangji@montclair.edu](mailto:wangji@montclair.edu)

## PDC Integration

The primary PDC intervention is the **Flag Maker** activity, a no-code activity
designed to introduce parallel computing concepts in an accessible CS1
context. The Fall 2025 course calendar identifies where this activity occurs
within the full course sequence. All files required to use the activity are
organized together in
[`Unplugged_Activities/FlagMaker/`](Unplugged_Activities/FlagMaker/).

## Student Privacy and Data Sharing

No identifiable student data should be included in this public repository.
Evaluation materials are limited to anonymized or aggregate data and
public-shareable analysis outputs. When student-level data cannot be shared
because of participant-privacy or IRB restrictions, the repository provides
aggregate summaries or a note explaining the restriction.
