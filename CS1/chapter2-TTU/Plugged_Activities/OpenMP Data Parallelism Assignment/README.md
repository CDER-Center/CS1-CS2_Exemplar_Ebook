# Plugged-In OpenMP Data Parallelism Assignment

In this CS1 activity, students compare sequential and parallel programs while exploring data parallelism with OpenMP.

## Activity Overview

| **Course level** | CS1 |
| **Estimated time** | Approximately 1-2 hours |
