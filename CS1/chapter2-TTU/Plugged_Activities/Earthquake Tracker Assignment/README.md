# Plugged-In Earthquake Tracker Assignment

In this CS1 activity, students develop an earthquake-tracking program while exploring distributed computing, remote services, and APIs.

## Activity Overview

| Field | Details |
| --- | --- |
| **Activity type** | Plugged |
| **Course level** | CS1 |
| **Estimated time** | Approximately 2-4 hours |
| **PDC topics** | Distributed computing, remote services, and APIs |
| **Related e-book section** | Section 3.4.1: Plugged-In Earthquake Tracker Assignment |
