This directory contains the resources for Physical Computing 2 Buttons and Multiple Processes.  

Physical Computing Project 2_Buttons and multiple processes.pdf contains the student instructions

button.py is the introductory sample code given to students that demonstrates some code involving a button

multiproc_intro.py is the introductory sample code given to student that demonstrates the use of the Python multiprocessing package

sample_solution.py is an example of code that students might write to solve the assignment.
