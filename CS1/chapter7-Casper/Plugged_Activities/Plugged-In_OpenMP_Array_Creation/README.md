This directory contains the materials used in the Plugged-In_OpenMP_Array_Creation Activity.

Introduction to OpenMP in C++ and parallel computing.pdf  is the activity directions for students that walks them through the activity.

arraySmall.cpp is code that is the solution for Part 2 #19

solution.cpp is a solution to the Part 3 where students are using the <chrono> library to benchmark the serial and parallel code.
