Here is the code we wrote today. We talked about OurArrayList and converted it to use generics: 

public class OurArrayList< T> {
	private int num;
	private T[] vals;

	@SuppressWarnings("unchecked")
	public OurArrayList() {
		num = 0;
		vals = (T[]) new Object[3];
	}

	public void add(T item) {
		vals[num] = item;
		num++;
	}

	public int size() {
		return num;
	}

}

We didn't talk about it, but here is a class with a main method that that runs the methods: 

public class Main {
 
    public static void main(String[] args) {
        OurArrayList< String> list = new OurArrayList< String>();

        System.out.println("size of empty list: " + list.size());

        list.add("Hello");
        System.out.println("size after adding something: " + list.size());
    }
}
